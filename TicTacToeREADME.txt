Instructions to run the project:
* Upload the zipped into blue
* Unzip the zipped folder
* Change directory to the unzipped folder
* Run make in terminal
* Run ./ticTacToe.x in terminal

* When the executable is initally running it will ask if the user wants to play against the AI
* If AI is chosen, you will have a series of questions relating to what rules to play with.
* Otherwise it will run the probabilites of both variations and it's sub variations.
